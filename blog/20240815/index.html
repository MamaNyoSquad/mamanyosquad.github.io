We're archiving!

published 15 July 2024 UTC

As you might have figured based on the title, we have decided that this website will be put in an archived state starting 15 Aug 2024! This is to focus on developing the next version of the website which is scheduled to debut on the Squad's 4th Inception Anniversary -- 6 Dec 2024.

It has long since dormant since I was focusing on other webdev projects in order to determine which design language is good for each context -- since I learned that most of the websites I maintain are using the same language as this website's upcoming version.

In response to this development, I've decided to put the site on hold, meaning that all planned changes to be implemented are to be thrown out of the window. Don't worry -- existing elements, including Blogs like this one -- will carry over to the new version!

So, what can we expect from the new version of the website?

  • Updated design language
  • 'Single-navigational' approach
  • Minimal browser cookie implementation
  • Updated look of Blog posts and Maintenance Tracker
  • and many more!

You can expect the full details to be disclosed during the 4th Inception Anniversary.

For now, let's hope for the success that is unveiling the new look of our new website -- based on version 20240518!

For more updates, you can keep us on the loop on X at @MamaNyoSquad.
